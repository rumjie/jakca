# JAKCA - 작업하기 좋은 카페 찾기

## Project info

- 기간 : 2025.07.01 ~ 
- 목적 : 공부나 작업하러 가기 좋은 카페를 찾아 헤매는 시간을 줄이자!

## Usage
1. 접속하여 위치 정보 제공 허용 (필수) 
2. 작카 리스트를 보고 카페 고르기   
    2.1 추천된 카페가 마음에 들었다면 성공!   
    2.2 추천된 카페가 마음에 들지 않았다면 리뷰 쓰기 (서비스 품질 향상에 큰 도움이 됩니다..💕)

## Technology
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
